# Native API reference

The native reference is generated by Rustdoc with dependencies excluded and private items included:

```powershell
cargo doc --manifest-path src-tauri/Cargo.toml --no-deps --document-private-items
```

[Open the assembled native API reference](../api/vanehub_ai_lib/index.html).

The reference exposes internal ownership and contract documentation for contributors. It does not make crate-private items a supported external library API, and implementation visibility is not widened for documentation.

The selected documentation-boundary inventory lives in `docs/developer-guide/native-boundaries.json`.

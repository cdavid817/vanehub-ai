# VaneHub AI 用户指南

<a href="../en/index.html">English</a>

本指南从验证 AI Coding CLI 开始，带你创建一个可工作的 VaneHub AI 会话。

## 状态标签

- **已交付** —— 用户可见路径已经实现并经过验证。
- **预览** —— Service 或 mock contract 已存在，但正常产品工作流尚未完成。
- **仅 Web/mock** —— 确定性的浏览器行为，不会产生 native side effect。
- **仅桌面端** —— 依赖 Tauri runtime 与本地操作系统访问能力。
- **规划中** —— 当前尚不可用。

从[安装并认证 CLI](getting-started.md)开始。

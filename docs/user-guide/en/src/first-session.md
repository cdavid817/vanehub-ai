# Create your first session

**Status: Delivered — desktop and Web/mock with different side effects.**

1. Open VaneHub AI.
2. Select **New**.
3. Choose **Single Agent**.
4. Select an available Agent.
5. Choose a project directory and provide a session title.
6. Create the session and use the Workspace terminal.

![English create-session dialog using synthetic VaneHub Demo project data](../assets/screenshots/create-session-en.png)

On desktop, the project path represents a real directory and Agent execution uses the selected installed CLI. In Web/mock, the same flow uses synthetic state and a simulated terminal; no local process or SQLite database is changed.

The currently disabled **Multi Agent** option is not part of this delivered workflow. See the [Multi-Agent preview](multi-agent-workflow.md).

# Support

VaneHub AI is community-supported.

- Search [existing issues](https://github.com/cdavid817/vanehub-ai/issues) before opening a new one.
- Use the bug form for reproducible defects and the feature form for product proposals.
- Include the application version, operating system, relevant agent CLI versions, and minimal reproduction steps.
- Redact tokens, account details, local paths, conversation content, and other sensitive information from logs and screenshots.

For security vulnerabilities, follow [SECURITY.md](SECURITY.md) and report privately. Public issues are not an appropriate security-reporting channel.
